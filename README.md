# Whimsical fonts

This repo contains the free fonts used in the [Whimsical whiteboard app](https://whimsical.com).

These fonts are licensed under the Open Font License, allowing them to be used, studied, modified and
redistributed freely as long as they are not sold by themselves.

| Font | Description |
|---------|-------------|
| **[Dinsical](dinsical)** | The main font used in Whimsical. Available in predefined weights as well as a variable font. 
| **[Monsical](monsical)** | The monospaced font used in Whimsical |
